1 # DS1302 RTC (Real Time Clock) library for Arduino
2 
3 [![Build Status](https://travis-ci.org/Erriez/ErriezDS1302.svg?branch=master)](https://travis-ci.org/Erriez/ErriezDS1302)
4 
5 This is a 3-wire DS1302 RTC (Real Time Clock) library for Arduino.
6 
7 ![DS1302 RTC](https://raw.githubusercontent.com/Erriez/DS1302/master/extras/DS1302.png)
8 
9 ## Library features
10 
11 * libc `<time.h>` compatible
12 * Read/write date/time `struct tm`
13 * Set/get Unix epoch UTC `time_t`
14 * Set/get time (hours, minutes, seconds)
15 * Set/get date and time (hour, min, sec, mday, mon, year, wday)
16 * Read / write 31 Bytes battery backupped RTC RAM.
17 * Programmable trickle charge to charge super-caps / lithium batteries.
18 * Optimized IO interface for Atmel AVR platform.
19 
20 ## DS1302 specifications
21 
22 **IMPORTANT NOTES:**
23 
24 * The DS1302 RTC time may deviate >1 minute each day, so this device is not recommended for designs with high precision requirements.
25 * The [high precision DS3231 I2C RTC](https://github.com/Erriez/ErriezDS3231) is recommended for new designs.
26 * The 3-wire interface is **NOT** compatible with SPI.

51 ## Usage
52 
53 **Initialization**
54 
55 ```c++
56 #include <ErriezDS1302.h>
57 
58 // Connect DS1302 data pin to Arduino DIGITAL pin
59 #if defined(ARDUINO_ARCH_AVR)
60 #define DS1302_CLK_PIN 2
61 #define DS1302_IO_PIN 3
62 #define DS1302_CE_PIN 4
63 #elif defined(ARDUINO_ARCH_ESP8266)
64 #define DS1302_CLK_PIN D4
65 #define DS1302_IO_PIN D3
66 #define DS1302_CE_PIN D2
67 #elif defined(ARDUINO_ARCH_ESP32)
68 #define DS1302_CLK_PIN 0
69 #define DS1302_IO_PIN 4
70 #define DS1302_CE_PIN 5
71 #else
72 #error #error "May work, but not tested on this target"
73 #endif
74 
75 // Create RTC object
76 ErriezDS1302 rtc = ErriezDS1302(DS1302_CLK_PIN, DS1302_IO_PIN, DS1302_CE_PIN);
77 
78 void setup()
79 {
80  // Initialize RTC
81  while (!rtc.begin()) {
82  Serial.println(F("RTC not found"));
83  delay(3000);
84  }
85 }
86 ```
87 
88 **Check oscillator status at startup**
89 
90 ```c++
91 // Check oscillator status
92 if (!rtc.isRunning()) {
93  // Error: RTC oscillator stopped. Date/time cannot be trusted.
94  // Set new date/time before reading date/time.
95 
96  // Enable oscillator
97  rtc.clockEnable(true);
98 }
99 ```
100 
101 **Set time**
102 
103 ```c++
104 // Write time to RTC
105 if (!rtc.setTime(12, 0, 0)) {
106  // Error: Set time failed
107 }
108 ```
109 
110 **Get time**
111 
112 ```c++
113 uint8_t hour;
114 uint8_t minute;
115 uint8_t second;
116 
117 // Read time from RTC
118 if (!rtc.getTime(&hour, &minute, &second)) {
119  // Error: RTC read failed
120 }
121 ```
122 
123 **Set date and time**
124 
125 ```c++
126 // Write RTC date/time: 13:45:09 31 December 2019 0=Sunday, 2=Tuesday
127 if (!rtc.setDateTime(13, 45, 9, 31, 12, 2019, 2) {
128  // Error: RTC write failed
129 }
130 ```
131 
132 **Get date and time**
133 
134 ```c++
135 uint8_t hour;
136 uint8_t min;
137 uint8_t sec;
138 uint8_t mday;
139 uint8_t mon;
140 uint16_t year;
141 uint8_t wday;
142 
143 // Read RTC date/time
144 if (!rtc.getDateTime(&hour, &min, &sec, &mday, &mon, &year, &wday) {
145  // Error: RTC read failed
146 }
147 
148 // hour: 0..23
149 // min: 0..59
150 // sec: 0..59
151 // mday: 1..31
152 // mon: 1..12
153 // year: 2000..2099
154 // wday: 0..6 (0=Sunday .. 6=Saturday)
155 ```
156 
157 **Write date/time struct tm**
158 
159 ```c++
160 struct tm dt;
161 
162 dt.tm_hour = 12;
163 dt.tm_min = 34;
164 dt.tm_sec = 56;
165 dt.tm_mday = 29;
166 dt.tm_mon = 1; // 0=January
167 dt.tm_year = 2020-1900;
168 dt.tm_wday = 6; // 0=Sunday
169 
170 if (!rtc.write(&dt)) {
171  // Error: RTC Read failed
172 }
173 ```

259 ## Pin configuration
260 
261 **Note:** ESP8266 pin D4 is high during a power cycle / reset / flashing which may corrupt RTC registers. For this reason, pins D2 and D4 are swapped.
262 
263 | DS1302 Pin | DS1302 IC | Atmel AVR | ESP8266 | ESP32 |
264 | :--------: | :--------: | :-------------: | :-----: | :---: |
265 | 4 | GND | GND | GND | GND |
266 | 8 | VCC2 | 5V (or 3.3V) | 3V3 | 3V3 |
267 | 7 | SCLK (CLK) | 2 (DIGITAL pin) | D4 | 0 |
268 | 6 | I/O (DAT) | 3 (DIGITAL pin) | D2 | 5 |
269 | 5 | CE (RST) | 4 (DIGITAL pin) | D2 | 4 |

271 ## API changes v1.0.0 to v2.0.0
272 
273 The API has been changed to make RTC libraries compatible with libc `time.h`. This makes it easier
274 to calculate with date/time and port the application to different platforms. See changes below:
275 
276 | v1.0.0 | v2.0.0 |
277 | -------------------------------* | -----------------------------------------------------------* |
278 | `DS1302_DateTime` | `struct tm` |
279 | | `clearOscillatorStopFlag()` merged into `clockEnable()` |
280 | `isHalted()` | `bool clockEnable(bool enable)` |
281 | `halt()` | `void isRunning()` |
282 | `setDateTime()` | `void write(struct tm *dt)` |
283 | `getDateTime()` | `bool read(struct tm *dt)` |
284 | `getEpochTime()` | `time_t getEpoch()` |
285 | | `void setEpoch(time_t t)` |
286 | `writeProtect()` | Removed |
287 | `isProtected()` | Removed |
288 | | `void setDateTime(uint8_t hour, uint8_t min, uint8_t sec, uint8_t mday, uint8_t mon, uint16_t year, uint8_t wday)` |
289 | | `void getDateTime(uint8_t *hour, uint8_t *min, uint8_t *sec, uint8_t *mday, uint8_t *mon, uint16_t *year, uint8_t *wday)` |
290 | `ErriezDS3231Debug` | class removed to reduce flash size |
